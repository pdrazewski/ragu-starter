# ragu-starter [![NPM version][npm-image]][npm-url]
> Generator for ragu.io components

## Installation

First, install [Yeoman](http://yeoman.io) and ragu-starter using [npm](https://www.npmjs.com/) (we assume you have pre-installed [node.js](https://nodejs.org/)).

```bash
npm install -g yo
npm install -g ragu-starter
```

Then generate your new project:

```bash
yo ragu-starter
```
